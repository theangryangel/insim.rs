Clockwork Carnage is a Live for Speed competition framework. Admins create sessions, drivers join, and the server handles the rest.

{# ── Game modes ── #}

Game Modes

Metronome

A rounds-based precision challenge. Each round, drivers race to match a target lap time as closely as possible — not the fastest, but the most accurate. Full contact permitted. An admin starts each round manually.

Format

Rounds

Scoring

Accuracy

Contact

Yes

Shortcut

An always-on time attack. Drivers race checkpoint-to-finish segments and compete for the fastest time on a persistent leaderboard. No rounds, no admin — just drive whenever you want.

Format

Open

Scoring

Fastest

Contact

No

{# ── How it works ── #}

How It Works

1

An admin creates a session

Pick a track, layout, game mode and configure the rules.

2

The session goes live

The server loads the track and the session begins accepting drivers.

3

Drivers compete

Laps are tracked automatically. Results and standings update in real time.

4

Results are published

Final standings, round breakdowns, and write-ups appear on the session page.
